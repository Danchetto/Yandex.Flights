# Yandex.Flights

## Установка
<code>
    npm i
</code>

## Запуск

Для сборки проекта необходимо ввести:
<code>
    npm run build
</code>

Для запуска бэкенд сервера и проекта:
<code>
    npm run start
</code>

**Открыть в браузере *localhost:8080***

Для запуска задания про счетчик (лежит в папке ./ticker):
<code>
    npm run tick
</code>
